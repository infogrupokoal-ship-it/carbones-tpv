# Manual Operativo AI & DevOps

Este proyecto está gestionado mediante una arquitectura Multi-Agente IA y flujos DevOps.

## Archivos Clave

- `AI_WORKFLOW.md`: Cómo interactúan las IAs.
- `AI_RULES.md`: Reglas absolutas de las IAs.
- `AI_AUTONOMY_POLICY.md`: Qué puede hacer la IA sola.
- `SECURITY_CHECKLIST.md`: Auditorías de seguridad obligatorias.
- `MONITORING.md`: Configuración de logs y healthchecks.
- `BACKUPS.md`: Cómo hacer copias de seguridad de datos crudos.

## Mantenimiento

Ejecuta `python tools/ai_agents/run_agent_cycle.py --project .` para simular un ciclo de validación.
