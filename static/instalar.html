⚙️ Instalador Final (Android)

Estás intentando configurar el auto-arranque del Cajón Registrador y la Impresora en esta tablet.

Paso 1: Pulsa el botón azul para copiar el código mágico al portapapeles de esta tablet.
pkill -f local_printer_bridge; cd ~/carbones-tpv || cd /storage/emulated/0/Download/carbones_y_pollos_tpv; git pull origin master; python local_printer_bridge.py
✅ ¡Código copiado!
Paso 2: Sal de aquí y abre la aplicación Termux (la pantalla negra).

Paso 3: Mantén pulsado con el dedo en la pantalla negra, dale a PEGAR (o Paste) y pulsa la tecla Enter (Intro) en el teclado de la tablet.

Paso 4: Verás que empiezan a salir letras. ¡Ya está encendido el motor!

Paso 5: Abre la app de Google Play y descarga "Termux:Boot". Ábrela solo una vez para darle permisos y olvídate para siempre. Ya puedes abrir tu TPV normal.
