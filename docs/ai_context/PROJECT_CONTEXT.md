# PROJECT_CONTEXT.md

- **Proyecto:** Carbones y Pollos TPV Enterprise
- **Negocio:** Carbones y Pollos La Granja S.L. Venta de pollos asados, arroces, menús y bebidas.
- **Usuarios Principales:** Cajeros, cocineros, administradores, clientes (vía kiosko B2C).
- **Flujos Principales:** 
  - Venta directa en mostrador (TPV).
  - Autopedido en Kiosko.
  - Gestión de inventario y cocina (KDS).
  - Dashboard de reportes.
- **Estado Actual:** Sistema operativo con integración de IA (Carbonito), telemetría, facturación (Stripe) y PWA.
- **Diferencias con GestiónKoal:** Carbones es un entorno retail/restauración de alta disponibilidad. GestiónKoal es multiservicios/reparaciones de campo. NUNCA cruzar datos de clientes, productos ni configuraciones.
