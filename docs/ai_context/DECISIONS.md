# DECISIONS.md

- **Decisiones Técnicas:** Frontend mantenido en vainilla (HTML/CSS/JS) para máxima compatibilidad e independencia de frameworks pesados. Uso de SQLite para simplicidad y portabilidad en TPV local.

- **Decisiones de Negocio:** Separación de interfaces para Caja, Cocina (KDS) y Kiosko (Autopedido). Integración de IA para recomendaciones (Carbonito).

- **Restricciones:** No cambiar a frameworks pesados (React/Angular) sin autorización. No unificar con GestiónKoal.
