1. **Confirmar Proyecto:** Verificar estar en carbones_y_pollos_tpv.

2. **Leer Contexto:** Leer AGENTS.md, PROJECT_CONTEXT.md, ARCHITECTURE.md.

3. **Revisar Tareas y Problemas:** Leer TASKS.md y KNOWN_ISSUES.md.

4. **Historial:** Leer CHANGELOG_AI.md.

5. **Planificación:** Crear un plan antes de tocar código.

6. **Ejecución:** Identificar archivos, hacer cambios atómicos y probar localmente.

7. **Documentación:** Actualizar CHANGELOG_AI.md y TASKS.md.

8. **Entrega:** Generar resumen final para el usuario.
